# Qeshm-Power-Plant-Forecasting
# Gas Turbine Power Plant Load Forecasting

This project implements an Artificial Neural Network (ANN) model to predict the electricity generation load of gas units in the **Qeshm Combined Cycle Power Plant**. Accurate load forecasting is critical for optimizing generation scheduling and improving market participation.

## 📊 Project Overview
- **Objective:** Predicting power output based on historical operational data.
- **Methodology:** Time-series forecasting using Artificial Neural Networks (ANN).
- **Key Features:**
    - Data Preprocessing & Feature Engineering
    - Network Training & Validation
    - Performance metrics (MSE, MAE, R-squared)
